Advanced Replacement Cache

Using Nimrod Megiddo and Dharmendra S. Modha's paper, I developed an implementation of Advanced Replacement Cache. This technique combines both LRU and LFU caches, and uses the number of hits in each cache to determine what the optimal ratio of entries each of the caches should have.
