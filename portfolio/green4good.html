Green For Good

Problem: Develop a website that addressed the climate crisis

Demo

Our Approach

First, decide on the design.

We considered what it meant to address the climate crisis. We wrote different ways to take action, linked to different organizations, created awareness via reliable news sources.

We built out a wireframe. We created a design that consisted of different cards on the various pages that we would link. These cards would be easy to copy and paste into different places. We also looked at CSS templates and found one that we thought embodied the vibe associated with environmental justice.

Then, we developed different pages, working together to create them.

I focused mainly on the home page, the navigation bar, and the petition related pages. This included research for photos, petitions, different links etc, in addition to coding html/css/js.

We built it and presented it!

After the sprint, we had a showcase event with our families and prudential staff. A year and a half later, the Girls Who Code organization wanted to showcase it at a crossover event with AOC and Girls Who Code Founder Reshma Saujani.

Changes + Extensions for the Future

- Clean up code base — we didn’t know what we were doing so a lot of it was copy and paste from different OS code sources.

- Work on optimizing loading - The photos are high quality which eventually leads to large loading times.

- Create a tech stack - This would allow us to have news feeds, petition feeds, etc., as we had originally wanted, without having to manually update everything in the github repository.
